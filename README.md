## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/instagram-fe.git && cd instagram-fe && npm install
```

### Executing program

```
npm start
```
